▸ setConfig

Configure the line chart

setConfig
Parameters
config (object) A config object.
Name Description
config.keyType string Data type of x values: time, number, string
config.chartType string The type of marks: line, area, stackedArea
config.width number Outer chart width, including margins
config.height number Outer chart height, including margins
config.margin object Margin object {top, right, bottom, left}
config.grid string Grid visibility: horizontal, vertical, full
config.xAxisFormat string d3-format string for x axis
config.yAxisFormat string d3-format string for y axis
config.yTicks string Suggestion for the number of y ticks
config.colorSchema Array<string> List of colors to use on marks
config.tickSkip number The number of ticks to skip to simplify the x axis
Returns
object: The line chart instance.
Example
mapd3.Line(document.querySelector('.chart'))
.setConfig({
    width: 100
})
